# Interpretable Short-Term Electric Load Forecasting

This repository contains the main code associated with the article "Interpretable Short-Term Electric Load Forecasting" by Alessandro Nicola, Giorgia Ghione, Vincenzo Randazzo, Vincenzo Franzitta, Marco Badami, Eros Pasero, and Giansalvo Cirrincione. 

A sample anonymized dataset that mirrors the structure, features, and dimensions of the original dataset is provided.
